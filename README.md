# Kopac Advisory — Strategic Blue Theme V4

A responsive static website built with plain HTML, CSS and JavaScript.

## Open locally

1. Open this folder in Visual Studio Code.
2. Install the **Live Server** extension if needed.
3. Right-click `index.html` and choose **Open with Live Server**.

## V4 changes

- Clicking the KOPAC wordmark now smoothly returns to the true top of the page.
- The blue navigation underline draws more slowly.
- Section 2 practice-card transitions are substantially slower and calmer.
- Section 3 image-backed principle cards now have a restrained hover lift, image drift and shadow.
- Section 4 has been rebalanced so the written CPS 230 brief receives more width and the blue visual is smaller.
- Section headings across the site have been reduced.

## Files

- `index.html` — page content and structure
- `styles.css` — responsive design and animations
- `script.js` — menu, scroll effects, practice-card interactions and form behaviour
- `assets/` — local SVG illustrations

## Publishing

The form includes Netlify-compatible attributes. After deploying the folder to Netlify, form submissions can appear in the site's Forms panel.
